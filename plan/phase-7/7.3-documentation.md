# FocusDragon

**Powerful website and application blocker for macOS**

Block distracting websites and apps with advanced lock mechanisms to maintain focus and boost productivity.

## Features

- **Website Blocking** - DNS-level blocking via /etc/hosts
- **App Blocking** - Automatic termination of blocked applications
- **Lock Mechanisms** - Timer, Random Text, Schedule, and Restart locks
- **Browser Extensions** - Chrome, Firefox, Safari support
- **Statistics** - Track focus time and productivity streaks
- **Pomodoro Timer** - Built-in Pomodoro technique support

## Installation

1. Download FocusDragon-X.X.X.dmg
2. Open DMG and drag FocusDragon to Applications
3. Launch FocusDragon
4. Grant required permissions
5. Install browser extensions (optional)

## Requirements

- macOS 11.0 or later
- Administrator access (for hosts file modification)
- Accessibility permissions (for app monitoring)

## Usage

### Basic Blocking

1. Add websites or apps to block list
2. Click "Start Blocking"
3. Enter admin password when prompted
4. Blocked sites/apps will be inaccessible

### Lock Mechanisms

**Timer Lock** - Require countdown before unlock
**Random Text Lock** - Type random code to unlock
**Schedule Lock** - Auto-block during specific times
**Restart Lock** - Require system restarts to unlock

## Getting Started

### First Launch

On first launch, FocusDragon will request:
- **Administrator access** - Required to modify /etc/hosts
- **Accessibility permissions** - Required for app monitoring
- **Notifications** - Optional, for alerts

### Adding Sites to Block

1. Click "Add Website"
2. Enter domain (e.g., "youtube.com")
3. Click Add
4. Toggle site on/off as needed

### Starting a Block

1. Enable sites/apps to block
2. Click "Start Blocking"
3. Enter admin password
4. Blocking is now active

## Lock Mechanisms

### Timer Lock

Prevents stopping block until timer expires.

**Setup:**
1. Set duration (1 min - 24 hours)
2. Click "Start Timer Lock"
3. Cannot stop until time expires

**Use case:** Deep work sessions

### Random Text Lock

Requires typing random code to unlock.

**Setup:**
1. Click "Random Text Lock"
2. Random code displayed
3. Type exact code to unlock
4. Maximum 5 attempts

**Use case:** Prevent impulsive unblocking

### Schedule Lock

Auto-blocks during specific times.

**Setup:**
1. Create schedule (e.g., "Work Hours")
2. Set time range (9 AM - 5 PM)
3. Select days (Monday-Friday)
4. Enable schedule

**Use case:** Daily routines

### Restart Lock

Requires system restarts to unlock.

**Setup:**
1. Select number of restarts (1-10)
2. Activate lock
3. Must restart N times to unlock

**Use case:** Ultimate commitment

## Browser Extensions

### Chrome

1. Download extension from Chrome Web Store
2. Enable extension
3. Extension syncs with app automatically

### Firefox

1. Download from Firefox Add-ons
2. Install and enable
3. Configure permissions

### Safari

1. Extension bundled with app
2. Safari → Preferences → Extensions
3. Enable FocusDragon

## Statistics

Track your focus journey:

- **Total Focus Time** - Cumulative blocked time
- **Sessions** - Number of blocking sessions
- **Streak** - Consecutive days with sessions
- **Charts** - Visual progress over time

## Troubleshooting

### Sites not blocked

1. Check if blocking is active
2. Verify site in block list
3. Clear browser cache
4. Flush DNS: `sudo dscacheutil -flushcache`

### Apps keep launching

1. Verify app in block list
2. Check process monitoring is active
3. Restart daemon

### Extension not working

1. Reload extension
2. Check native messaging host
3. Reinstall extension

### Permission errors

1. System Settings → Privacy & Security
2. Grant Accessibility permission
3. Grant Full Disk Access (if needed)

## Advanced

### Command Line

Check daemon status:
```bash
sudo launchctl list | grep focusdragon
```

Manual hosts file check:
```bash
cat /etc/hosts | grep FocusDragon
```

### Uninstall

1. Stop blocking
2. Quit app
3. Delete from Applications
4. Remove daemon: `sudo launchctl unload /Library/LaunchDaemons/com.focusdragon.daemon.plist`
5. Remove config: `sudo rm -rf "/Library/Application Support/FocusDragon"`
```
